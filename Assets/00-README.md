# La Ecoferia — Kit de Documentación · Sprint 1

Marketplace multimarca de productos sustentables del Alto Valle (Neuquén y Río Negro), Patagonia Argentina.

## Índice de documentos

| Doc | Contenido | Audiencia |
|---|---|---|
| `01-PRD.md` | Visión, identidad de marca, público, roles, mapa del sitio, requisitos funcionales | Todos |
| `02-DESIGN-SYSTEM.md` | Tokens completos "Patagonian Artisanal Organic", tipografía, componentes, migración a Tailwind v4 | Frontend |
| `03-ARQUITECTURA.md` | Monorepo pnpm, stack (React 19 / Hono / Drizzle / Neon / Better Auth), flujo de datos, ADRs | Fullstack |
| `04-MODELO-DATOS.md` | Esquema de tablas, constraint XOR de marcas, índices, plan de seeds | Backend |
| `05-SPRINT-1.md` | Sprint goal, backlog con estimaciones, DoD, riesgos | Todos |
| `La-Ecoferia-Sprint1.pdf` | Recopilación visual con imágenes de las maquetas | Presentación / referencia |

## Fuentes originales del proyecto

- `PRD` — requisitos del producto (base del doc 01)
- `DESIGN.md` + `code__5_.html` — sistema de diseño y tokens (base del doc 02)
- `code__3/7/9/15/21_.html` — maquetas navegables (ingreso, blog de marca, directorio/paneles admin)
- `screen*.png`, `styles.png`, `Logo.png`, `Feed1-4.png` — capturas y assets visuales

1. `01-PRD.md` → entender el producto
2. `05-SPRINT-1.md` → saber qué se construye ahora
3. `03-ARQUITECTURA.md` + `04-MODELO-DATOS.md` → montar la base
4. `02-DESIGN-SYSTEM.md` → construir la UI
