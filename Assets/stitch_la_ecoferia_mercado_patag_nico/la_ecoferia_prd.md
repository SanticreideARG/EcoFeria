# La Ecoferia - Product Requirements Document (PRD)

## Project Overview
"La Ecoferia" is a multi-brand marketplace/eshop for sustainable products from the Alto Valle and Patagonia region (Neuquén and Río Negro, Argentina). It connects local entrepreneurs with conscious consumers, focusing on regional crafts, food, and recycled goods.

## Brand Identity
- **Aesthetic:** Warm, artisanal, organic, eco-conscious yet modern. Inspired by producer fairs and mountain town markets.
- **Color Palette:**
    - Earth Greens (Forest, Sage)
    - Terracotta / Soft Brick
    - Light Wood / Kraft Paper textures
    - Bone White (Backgrounds)
    - Mustard / Warm accents (CTAs)
- **Typography:**
    - Titles: Serif or Slab with human character (warm, approachable).
    - Body: Neutral, legible Sans-serif.
- **Imagery:** Wood textures, kraft paper, artisanal product shots, Patagonian nature accents (mountains, forests).

## Design Principles
- **Mobile-First Strict:** Every design starts with the mobile version.
- **PWA Ready:** Consider offline states and splash screens.
- **Authenticity:** Avoid "Corporate Tech" aesthetics. Focus on the human and local element.

## User Roles
1. **Client:** Browse brands, shop, messaging, notifications, favorites.
2. **Seller:** Manage brand(s), products, brand mini-landing, orders.
3. **Admin:** Manage owned brands + general moderation + seller approval.

## Site Structure / User Journey
- Landing Page (Public)
- Brand Directory
- Brand Mini-landing (Profile + Mini-blog + Contact)
- General Blog
- Cultural Agenda
- Eshop (Catalog + Cart)
- Checkout
- Client Panel (Orders, Messages, Favorites)
- Seller Board (Dashboard for sellers)
- Admin Dashboard (Supervision)
